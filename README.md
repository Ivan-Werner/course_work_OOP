# Курсовой проект. ООП
## Описание

Проект представляет собой приложение, которое получает информацию
о вакансиях с платформы HeadHunter.ru. Сохраняет ее в файл и позволяет удобно работать
с ней.

## Логика проекта:
### Проект состоит из модулей:
* abstracts.py - модуль с абстрактными классами, от которых наследуются классы,
используемые в проекте
* api.py - модуль для получения информации о вакансиях по api с сайта
HeadHunter.ru
* vacancy.py - модуль для работы с вакансиями
* saver.py - модуль, позволяющий сохранять информацию в файле формата JSON
* utils.py - модуль с вспомогательными функциями
